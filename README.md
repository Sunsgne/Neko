# Neko 🐱

A TypeScript + Node.js project baseline.

## Requirements

- Node.js >= 20
- npm

## Getting started

```bash
# Install dependencies
npm install

# Run in watch mode (TypeScript executed directly via Node)
npm run dev

# Type-check the project
npm run typecheck

# Build to dist/
npm run build

# Run the compiled output
npm start

# Run tests
npm test
```

## Project structure

```
.
├── src/            # Source code
│   └── index.ts    # Entry point
├── test/           # Tests (node:test)
├── tsconfig.json   # TypeScript configuration
└── package.json    # Scripts and dependencies
```
